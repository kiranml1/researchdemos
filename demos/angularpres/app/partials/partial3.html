Scopes Concept

Scope is in a particular Context based evaluation
  • <div ng-app ></div> Scope is basically closed in this div
  • $rootScope is the higer level state of scope heirarchy, if we want to create new child scope we need $rootscope.$new
  • we would use Angular Batarang Chrome Extension for listing out the scope key pairs in the application
  • $watch is used for Watching any model changes $apply is used for propagate the model changed and evaluating the anything into the angular context function $apply(expr) {
    try {
    return $eval(expr);
    } catch (e) {
    $exceptionHandler(e);
    } finally {
    $root.$digest();
    }
    }
  • Expression are Evaluated in the based on the compilation link from the scope
  • Nested Scopes = Child Scope / Isolated Scope ( They are different ) - ( they are made with prototype inheritance)
  • In MVC Pattern we would say
    Model is Scope of Angular Framework
    Controller is we can say Controller which would hold the scope communication with view
    View is the Obvious DOM Part we see the angular framework
  • <li ng-repeat="name in names">{{name}} from {{department}}</li>
    ngRepeat creates a new scope for evaluation of value
  • Emiting & Broadcasting is a concept of push/firing up scoping in heirarchies
    Emting - Push to Upper Hirarchy Broadcasting - Push to Lower Hierarchy
  • Scope Life Cycle
    • Creation - $injector
    • Watcher Registration - $scope.watch()
    • Model Mutation - $scope.apply()
    • Model Observation - $scope.digest()
    • Scope Destruction - $scope.destroy()
  • Scope and Directives
    • In Directives, Scope watches the expression values with $watch to update the view
    • After that it will use $apply to to update completly to view
    • Compilation of scope comes mostly in creating child scopes with directives such as ngRepeat
  • $evalSync is such as such a synchronizer for putting to watching in digest loop with a queue of options
  • $on(name,listener) we can call "$emit(name)"
    Easy for Custom Events and Core Events links
    $on('customevetn',listener) and $on('click|keyup....',listner)
  • $watch and $watchCollection, which can be used watching the expression an list/array of expression values $id is the unique identification of a scope
